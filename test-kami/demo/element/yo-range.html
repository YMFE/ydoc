yo-range

 文档

单个滑块：

示例描述：

<div class="yo-range"> <div class="track"> <span class="thumb" style="-webkit-transform: translatex(.5rem);"></span> </div> </div>

两个滑块：

示例描述：

<div class="yo-range"> <div class="track"> <span class="thumb" style="-webkit-transform: translatex(.5rem);"></span> <span class="thumb" style="-webkit-transform: translatex(1.5rem);"></span> </div> </div>

带刻度：

    • 0
    • 100
    • 300
    • 500
    • 不限

示例描述：

<div class="yo-range"> <ul class="scale"> <li class="divide">0</li> <li class="divide" style="left: 25%;">100</li> <li class="divide" style="left: 50%;">300</li> <li class="divide" style="left: 75%;">500</li> <li class="divide">不限</li> </ul> <div class="track"> <span class="thumb" style="-webkit-transform: translatex(.5rem);"></span> <span class="thumb" style="-webkit-transform: translatex(1.5rem);"></span> </div> </div>
